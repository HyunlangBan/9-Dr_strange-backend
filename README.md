# Features

* users
   - 유저정보저장
   - 회원가입 / 로그인
     - 회원가입시 유효성 검사
     - 회원가입시 패스워드 암호화
     - 로그인시 JWT Access 토큰 발행
   - 로그인 상태인지 확인하는 데코레이터 함수
   - 마이페이지
   
* product_app
   - 상품 리스트 구현
   - 상품 상세정보 구현
   - 상품 검색 구현
   - 상품 베스트셀러 구현
   
* cart_app
   - 장바구니 구현
      - 장바구니 추가 / 보기 / 삭제
